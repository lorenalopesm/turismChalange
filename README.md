# 🧳 Local Turístico

> Projeto desenvolvido como desafio da formação **Full-stack** da Rocketseat.

Este repositório reúne um dos desafios práticos da formação **Full-stack** da Rocketseat. O projeto consiste em uma Landing Page sobre a cidade de **Busan**, desenvolvida com **HTML5** e **CSS3**, seguindo um layout proposto no Figma.

Além de praticar a construção de páginas estáticas, este projeto faz parte da minha jornada de aprendizado em desenvolvimento web. Todos os desafios da formação serão mantidos públicos para registrar minha evolução ao longo do curso e compor meu portfólio.

---

## 🚀 Sobre a formação

Este projeto faz parte da **Formação Full-stack** da Rocketseat.

Ao longo da trilha, estou estudando e desenvolvendo projetos envolvendo:

- Fundamentos da programação
- HTML5 e CSS3
- JavaScript
- Git e GitHub
- Interfaces dinâmicas e interativas
- React
- Node.js
- APIs
- Desenvolvimento Full-stack
- Boas práticas de desenvolvimento

Cada novo desafio representa uma oportunidade de aplicar novos conceitos e evoluir tanto na organização do código quanto na construção de interfaces.

---

## 📚 Objetivos deste projeto

Neste desafio, pratiquei conceitos como:

- Estrutura semântica com HTML5
- Organização de conteúdo utilizando `header`, `main`, `section` e `footer`
- Listas ordenadas e não ordenadas
- Inserção e estilização de imagens
- Tipografia com Google Fonts
- Espaçamentos e alinhamentos com CSS
- Pseudo-elementos e contadores em CSS
- Boas práticas na construção de páginas estáticas
